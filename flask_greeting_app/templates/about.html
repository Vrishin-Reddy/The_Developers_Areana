📚 About This Flask Application

This Flask application demonstrates core web development concepts including routing, form handling, template rendering, and dynamic content generation. It's designed as a learning project for Month 3: Introduction to Flask.

🌐 Flask Features Used

  • Routing: @app.route() decorators
  • HTTP Methods: GET and POST handling
  • Template Rendering: render_template()
  • Form Processing: request.form
  • URL Generation: url_for()
  • Redirects: redirect()

🎨 Jinja2 Template Features

  • Template Inheritance: extends and blocks
  • Variable Rendering: {%raw%}{{ variable }}{%endraw%}
  • Control Structures: {%raw%}{% if %}{%endraw%}, {%raw%}{% for %}{%endraw%}
  • Filters: Data transformation in templates
  • URL Generation: {%raw%}{{ url_for() }}{%endraw%} in templates
  • Form State: Preserving form data

✓ Jinja2 Templating

✓ Form Processing
